# SudokuSolver
My first python program

Uses brute force approach to solve a standard 9x9 sudoku board

takes a sudoku board as input, with 
EX:
  003020600
  900305001
  001806400
  008102900
  700000008
  006708200
  002609500
  800203009
  005010300
